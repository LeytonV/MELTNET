HEADACHE is a brutal FPS with roguelike elements, inspired by the ultraviolence of the Hotline Miami series. The goal of the project was to make the player think and act on their feet, encouraging improvisation and recklessness. Everything in the game was designed, programmed, modelled, animated, and generally created by me.

THE NITTY GRITTY

The game uses a texture-to-level system, which I explain about in the next section. As well as some interesting shaders for the artstyle. HEADACHE was more of a design challenge for me to create a first-person Hotline Miami, which is a challenge as Hotline Miami's whole gameplay loop, and the experience that comes from it, relies on the player being able to see top down, and see threats through walls thus being able to plan around them. HEADACHE resolves this by relying on the player's instincts, as well as using audio, and directional markers when surrounded by enemies.

THE LEVEL GENERATOR

Every level in HEADACHE is randomly generated. This is mainly to keep things replayable. The final level is assembled by creating a 3x3 grid, and creating a route for the level to follow (fig.1) Once that's complete, the level is created by generating rooms for each tile in the route. These rooms exist on the hard drive as 49x49 textures (fig.2), in which the game walks through, and based on the pixel's color, spawns walls, glass, doors, enemies, etc. (fig.3)

THE SHADERS

HEADACHE's artstyle required a lot of shaders to be written. The main ones are the dither shader, and the outline shader. The dither shader (which causes these pixel-ly effects over dark areas) was created by converting the pixel's color into a HSV color, then taking the Value of that HSV and running it through a Bayer dither algorithm to alternate the pixels based on the value. After that the final output color is a blend between black, and yellow, based on that value. The outlines are created by checking each pixel's surrounding pixels, and anywhere that the distance to each pixel's coordinate exceeds a certain value, is where a outline is drawn.
